Scrabble Tile Game

This Java project implements a tile-based word game inspired by Scrabble.

## Overview
The program works with character tiles, checks whether words can be formed from
a hand, calculates word scores, and identifies the highest-scoring valid word
from a dictionary.

## Features
- Tile-based word validation
- Word score calculation
- Best-word selection
- Character-array processing
- Dictionary-based search
