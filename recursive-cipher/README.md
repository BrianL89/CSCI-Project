## Recursive Cipher Decoder

This Java project implements a recursive algorithm to detect and extract words
embedded inside a cipher text.

## Overview
The program scans a character stream, identifies valid dictionary words,
removes discovered words from the cipher, and then recursively processes the
remaining text until no more words can be found.

## Features
- Recursive word extraction
- Character-array based string processing
- Dictionary-based word detection
- Dynamic storage of discovered words
