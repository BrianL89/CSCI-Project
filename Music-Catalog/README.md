## Music Catalog Data Structures

This Java project builds custom ArrayList and LinkedList implementations for a
music catalog application.

## Overview
The program supports catalog operations such as adding, removing, searching,
retrieving, and clearing songs. Both array-based and linked-list-based
implementations are developed and compared.

## Features
- Custom ArrayList implementation
- Custom LinkedList implementation
- Add and remove operations
- Search and retrieval functions
- Traversal and list management
- Comparison of data structure behavior

- Data structures
- Dynamic arrays
- Linked lists
- Traversal logic
- Algorithm efficiency
