## Sorting Algorithms and Performance Analysis

This Java project implements multiple sorting algorithms and compares their
performance on class schedule data.

## Overview
The project includes implementations of selection sort, bubble sort,
insertion sort, and quicksort. It also tracks operations such as comparisons,
swaps, and allocations to analyze algorithm efficiency.

## Features
- Selection sort implementation
- Bubble sort implementation
- Insertion sort implementation
- Quicksort implementation
- Performance profiling and comparison
- Analysis of algorithm efficiency
